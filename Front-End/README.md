*这是iWorking的小程序源码*   
### iWorking 是什么？

匿名的工作岗位交流平台，帮助求职者了解岗位内幕

它的功能：岗位薪酬、工作满意度、面试经验等
    
### 郑重声明：
iWorking不会收集用户的任何信息，这个平台的信息将由各位求职者共同维护
